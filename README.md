# azam.dev — Portfolio

Personal portfolio website built with Nuxt 3 and Tailwind CSS.

## Stack

- **Framework**: Nuxt 3
- **Styling**: Tailwind CSS with custom design tokens
- **Fonts**: Inter + Geist (via Google Fonts)
- **Icons**: Material Symbols Outlined
- **Contact form**: Formspree
- **Package manager**: Bun

## Development

```bash
bun install
bun run dev
```

## Build

```bash
bun run build
bun run preview
```

## Docker

```bash
docker build -t portfolio .
docker run -p 3000:3000 portfolio
```
